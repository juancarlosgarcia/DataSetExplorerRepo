Introduction

Quick Data Explorer makes easy to explore any dataset getting familiar with:

  • Row Data
  • Column Data Types

Also we can do interactive operations with the data:

  • Filter row data and filter by specific columns or data type.
  • Sort ascending, descending by any column.
  • Plot X,Y relations with facet and color support.
  • Analyze clusters using K-means.

Data Exploration

This version (1.0) supports only data frame. The first step is to pick the dataset, at this moment only standard datasets are supported.

After pick the data set will be used as a context for the plot and cluster panels. For instance let's pick iris. Also we can filter by specific columns or filter by data type

contextData<-iris
unlist(unique(lapply(contextData,class)))

## [1] "numeric" "factor"

Plot Exploration

The dataset iris has been selected as a context but also columns could be filtered in the data exploration section, this is useful for datasets with large amount of columns. Now we are ready for explore data using a plot.

The application allows to configure X,Y. Also supports Color and Facets. The following is a plot X=Sepal.Length, y=Sepal.Width, Color=Species,Facet='.~Species'.

Cluster Exploration

This feature allow to explore a cluster using the dataset in context (iris in our example) using k-means technique.

The application allows to configure, X,Y and the number of clusters that is a number between 1 and 9.
